
Javascript
```js
function printName(fName,lName){
    console.log(fName+" "+lName)
}
printName('Jonathan','Smith')
```

Python
```py
def print_name(f_name,l_name):
    print(f_name+" "+l_name)

print_name('Jonathan','Smith')
```

## differences
- : instead of {}
- "def" instead of "function"
- print Vs console.log
- snake case Vs camel case

## Data Types

| Python    | Javascipt |
|-----------|-----------|
| boolean   | boolean   |
| strings   | strings   |
| int, float| numbers   |
| lists     | array     |
| dictionary| object    |
| tuples    | none      |
